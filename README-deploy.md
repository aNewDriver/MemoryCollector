# Memory Collector Demo

《记忆回收者》手机试玩版 - Vercel部署配置

## 项目结构
- `mobile-demo/` - 游戏主目录
- `vercel.json` - Vercel部署配置

## 游戏特性
- 500关完整配置
- 800+张卡牌
- 完整战斗系统
- 剧情模式
- 存档系统
- 成就系统
- 排行榜

## 技术栈
- 纯HTML/CSS/JavaScript
- 无需构建工具
- 支持所有现代浏览器

## 本地运行
```bash
cd mobile-demo
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 在线试玩
部署后可通过Vercel URL直接访问。
