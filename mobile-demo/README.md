# 记忆回收者 - 手机试玩版

## 🎮 游戏说明

### 核心玩法
1. **抽卡**：点击"抽卡"消耗100金币获得随机卡牌
2. **收集**：积累至少3张卡牌才能进入战斗
3. **战斗**：选择卡牌攻击敌人，利用元素克制造成更高伤害

### 元素克制关系
```
火 → 金 → 木 → 土 → 水 → 火
光 ↔ 暗（互克）
```
克制时伤害 +30%，被克时伤害 -30%

### 稀有度
- N (普通) - 灰色
- R (稀有) - 蓝色
- SR (史诗) - 紫色
- SSR (传说) - 金色
- UR (神话) - 红色

## 🚀 部署指南

### 方式1：Vercel（推荐）
1. 访问 https://vercel.com
2. 导入 GitHub/Gitee 仓库
3. 设置输出目录为 `mobile-demo`
4. 自动获得 https 链接

### 方式2：Netlify
1. 访问 https://netlify.com
2. 拖拽 `mobile-demo` 文件夹上传
3. 立即获得访问链接

### 方式3：Gitee Pages
1. 在 Gitee 仓库设置中开启 Pages
2. 选择 `mobile-demo` 目录
3. 获得 https://用户名.gitee.io/仓库名/mobile-demo/
