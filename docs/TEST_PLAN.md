## 二、性能测试

### 测试环境
- 设备: 中端手机 (如 iPhone 12 / 小米12)
- 网络: 4G/WiFi

### 测试项
| 测试项 | 标准 | 测试方法 |
|--------|------|----------|
| 启动时间 | < 5秒 | 记录点击图标到主界面出现 |
| 场景切换 | < 1秒 | 记录点击按钮到新场景渲染完成 |
| 战斗帧率 | >= 30fps | 使用Cocos Profiler监测 |
| 内存占用 | < 500MB | 监测内存峰值 |
| 包体大小 | < 100MB | 查看构建产物 |
| 发热控制 | 无明显发热 | 连续游玩30分钟 |

## 七、自动化测试建议

### 可自动化场景
1. **登录流程** - 使用脚本模拟登录
2. **战斗数值** - 编写单元测试验证公式
