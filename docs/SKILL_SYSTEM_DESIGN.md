# 记忆回收者 - 卡牌技能系统设计文档

## 1. 技能系统概述

### 1.1 技能类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **主动技能** | 战斗中手动释放 | 火球术、治疗术 |
| **被动技能** | 自动生效 | 暴击加成、生命恢复 |
| **连携技能** | 特定卡牌组合触发 | 火+木=燃烧 |
| **觉醒技能** | 觉醒后解锁 | 终极奥义 |

### 1.2 技能触发时机

```
战斗开始时    → 被动技能生效
回合开始时    → 持续效果结算
攻击时        → 攻击前/攻击后技能
被攻击时      → 受击前/受击后技能
击败敌人时    → 击杀技能
回合结束时    → 结束阶段技能
```

## 2. 技能效果分类

### 2.1 伤害类

```typescript
interface DamageEffect {
  type: 'damage';
  value: number;           // 基础伤害值
  target: 'single' | 'all' | 'random';  // 目标类型
  element?: ElementType;   // 元素属性
  isPierce?: boolean;      // 是否穿透防御
  isCrit?: boolean;        // 是否可暴击
}
```

**示例技能：**
- `烈焰斩`：对单体造成150%攻击力的火属性伤害
- `暴风雪`：对全体造成80%攻击力的水属性伤害
- `雷霆一击`：对随机3个敌人造成120%攻击力的雷属性伤害

### 2.2 治疗类

```typescript
interface HealEffect {
  type: 'heal';
  value: number;           // 治疗量
  target: 'self' | 'single' | 'all';
  isPercent?: boolean;     // 是否为百分比治疗
}
```

**示例技能：**
- `治愈之光`：恢复单体友方30%最大生命值
- `群体治疗`：恢复全体友方15%最大生命值
- `生命绽放`：每回合开始时恢复自身10%生命值（持续3回合）

### 2.3 增益类（Buff）

```typescript
interface BuffEffect {
  type: 'buff';
  stat: 'atk' | 'def' | 'spd' | 'crit' | 'hp';
  value: number;
  duration: number;        // 持续回合数
  isPercent?: boolean;
}
```

**示例技能：**
- `战意沸腾`：攻击力+30%，持续2回合
- `钢铁意志`：防御力+50%，持续3回合
- `风之祝福`：速度+20%，持续整场战斗

### 2.4 减益类（Debuff）

```typescript
interface DebuffEffect {
  type: 'debuff';
  stat: 'atk' | 'def' | 'spd' | 'crit' | 'heal';
  value: number;
  duration: number;
  isPercent?: boolean;
}
```

**示例技能：**
- `破甲`：降低目标50%防御，持续2回合
- `虚弱`：降低目标30%攻击，持续2回合
- `冰冻`：降低目标100%速度，持续1回合（跳过下回合）

### 2.5 控制类

```typescript
interface ControlEffect {
  type: 'control';
  effect: 'stun' | 'silence' | 'freeze' | 'sleep';
  duration: number;
  chance: number;          // 触发概率
}
```

**示例技能：**
- `眩晕打击`：50%概率眩晕目标1回合
- `沉默术`：使目标无法使用技能，持续2回合
- `深度睡眠`：使目标沉睡2回合（受到伤害醒来）

### 2.6 特殊类

```typescript
interface SpecialEffect {
  type: 'special';
  effect: 'revive' | 'shield' | 'steal' | 'reflect' | 'invincible';
  value?: number;
  duration?: number;
}
```

**示例技能：**
- `复活术`：复活一名阵亡友方，恢复30%生命值
- `护盾`：为目标添加可吸收500点伤害的护盾
- `伤害反射`：反射30%受到伤害给攻击者，持续2回合
- `无敌`：免疫所有伤害，持续1回合

## 5. 觉醒技能设计

### 5.1 觉醒条件

- 卡牌达到最高星级（5星/6星）
- 完成特定觉醒任务
- 消耗觉醒材料

### 5.2 觉醒技能示例

| 角色 | 觉醒技能 | 效果 |
|------|----------|------|
| 烬羽 | 焚天灭地 | 对全体敌人造成500%火属性伤害，并使其燃烧3回合 |
| 青漪 | 水之洗礼 | 全体友方恢复50%生命值，并获得2回合无敌 |
| 银锋 | 万剑归宗 | 对单体造成1000%伤害，无视防御和闪避 |

## 6. 技能升级系统

### 6.1 升级消耗

| 等级 | 消耗金币 | 消耗材料 |
|------|----------|----------|
| 1→2 | 1000 | 技能书×1 |
| 2→3 | 3000 | 技能书×3 |
| 3→4 | 6000 | 高级技能书×1 |
| 4→5 | 10000 | 高级技能书×3 |

### 6.2 升级效果

每升1级，技能效果提升10%
- 伤害类：伤害+10%
- 治疗类：治疗量+10%
- Buff/Debuff：持续时间+1回合或效果+10%

## 7. 技能平衡性原则

1. **冷却与强度平衡**：强力技能应有较长冷却
2. **元素克制放大**：鼓励使用克制元素
3. **职业定位清晰**：战士输出、法师AOE、治疗辅助、刺客暴击、坦克承伤
4. **连携奖励**：多元素组合应有明显优势
5. **反制机制**：控制技能应有对应的解除手段
