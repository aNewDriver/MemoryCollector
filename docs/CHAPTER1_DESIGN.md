# 记忆回收者 - 剧情模式第一章设计

## 第一章：觉醒 (The Awakening)

### 故事背景
玩家扮演一名"记忆回收者"——一种能够进入他人潜意识、收集散落的记忆碎片的特殊存在。在崩坏的世界中，人们的记忆正在不断流失，而玩家的使命就是回收这些记忆，重构失落的过去。

---

## 场景设计

### 场景1：梦境入口 (Dream Entrance)

**视觉效果：**
- 朦胧的雾气环绕
- 漂浮的光点（记忆碎片的前兆）
- 远处隐约可见的记忆之门

**对话/旁白：**
```
[开场动画]
画面：黑暗中，一束光逐渐亮起...

旁白："又是一个梦境。"
旁白："在这里，记忆是实体，情感是力量。"
旁白："而我，是回收这些失落记忆的人。"

[出现主角立绘]
主角："这次的委托...是关于一个失去童年的孩子。"
主角："让我看看，他的记忆里藏着什么..."

[进入新手引导]
```

**新手引导流程：**
1. 点击屏幕移动主角
2. 靠近光点收集第一块记忆碎片
3. 遇到第一个敌人（记忆怪物）
4. 触发战斗教学

---

### 场景2：初次战斗 (First Battle)

**战斗前对话：**
```
主角："这是...记忆的守卫？"
主角："看来想要回收记忆，必须先击败它们。"

系统提示：「点击卡牌进行攻击」
系统提示：「使用技能可以造成更多伤害」
```

**战斗配置：**
- 敌人：记忆守卫 x1 (弱化版)
- 玩家：初始卡组（3张N卡）
- 教学提示高亮显示

**战斗后：**
```
主角："比想象中轻松。"
主角："收集到第一块记忆碎片了...这是一段关于生日的记忆。"
```

---

### 场景3：记忆重构 (Memory Reconstruction)

**视觉效果：**
- 收集到的记忆碎片在屏幕中央旋转
- 逐渐拼合成一幅画面

**剧情展示：**
```
[记忆画面显现]
画面：一个小男孩在吹蜡烛，周围是模糊的人影...

主角："这是...他的五岁生日？"
主角："为什么这段记忆会失落..."

[发现隐藏信息]
主角："等等，画面角落里那个黑影是...？"
```

**解锁功能：**
- 记忆图鉴系统解锁
- 可以查看已收集的记忆

---

### 场景4：Boss战前 (Before Boss)

**场景变化：**
- 雾气变得更浓
- 背景音乐变得紧张

**对话：**
```
主角："前面的记忆波动很强烈..."
主角："应该是这段记忆的核心守卫。"

[出现Boss剪影]
主角："看来要有一场硬仗了。"
```

---

### 场景5：第一章Boss战 (Chapter 1 Boss)

**Boss设计：**
- 名称：「遗忘守护者」
- 外观：由破碎的记忆碎片组成的巨大人形
- 元素：暗属性
- 特殊机制：每3回合释放「记忆抹除」（全体伤害+清除玩家Buff）

**战前对话：**
```
Boss："又是回收者...你们这些窃取记忆的小偷！"
主角："不，我是来拯救这些记忆的。"
Boss："可笑！记忆一旦破碎，就永远无法复原！"
主角："那我就用行动证明给你看。"
```

**战斗流程：**
1. 第1-2回合：正常攻击
2. 第3回合：Boss蓄力（玩家可以看到提示）
3. 第4回合：Boss释放「记忆抹除」
4. 教学：使用治疗技能或防御技能应对

**胜利后：**
```
Boss："不可能...你竟然..."
主角："记忆的力量，来源于想要记住的意志。"
主角："这就是为什么我能赢。"

[Boss消散，大量记忆碎片飞出]
```
