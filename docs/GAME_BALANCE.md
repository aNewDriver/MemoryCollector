## 一、基础数值体系

### 1.1 角色基础属性

| 属性 | 说明 | 计算公式 | 备注 |
|------|------|---------|------|
| 生命值(HP) | 角色生命 | 基础1000 + 等级×100 + 装备加成 | 战斗归零即死亡 |
| 攻击力(ATK) | 伤害输出 | 基础100 + 等级×10 + 装备加成 | 影响技能伤害 |
| 防御力(DEF) | 伤害减免 | 基础50 + 等级×5 + 装备加成 | 防御公式: 伤害 = 原始伤害 × 100/(100+防御) |
| 速度(SPD) | 行动顺序 | 基础100 + 等级×2 + 装备加成 | 速度高者先行动 |
| 暴击率(CRIT) | 暴击概率 | 基础5% + 装备/技能加成 | 上限75% |
| 暴击伤害(CRIT DMG) | 暴击倍率 | 基础150% + 装备加成 | 上限300% |
| 命中率(HIT) | 命中概率 | 基础100% - 对方闪避率 | 最低5% |
| 闪避率(EVA) | 闪避概率 | 基础5% + 装备/技能加成 | 上限50% |
