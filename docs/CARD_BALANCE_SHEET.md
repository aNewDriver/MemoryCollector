## 6. 战斗数值平衡

### 6.1 伤害公式

```
最终伤害 = (攻击者ATK × 技能倍率 - 防御者DEF × 0.5) × 克制系数 × 暴击系数 × 随机波动(0.9-1.1)

最小伤害 = 1
```

## 7. 数值验证检查表

### 7.1 平衡性检查

- [ ] 各稀有度卡牌DPS差距在合理范围（±20%）
- [ ] 各职业定位清晰（战士输出、法师AOE、刺客爆发、坦克承伤、治疗辅助）
- [ ] 元素克制效果明显但不至于碾压
- [ ] 高星低稀有度 vs 低星高稀有度有取舍空间

### 7.2 经济性检查

- [ ] 零氪玩家每月可获得足够资源培养1-2张SSR
- [ ] 月卡玩家进度提升约30%
- [ ] 重氪玩家不能通过数值碾压形成绝对优势
- [ ] 资源回收机制（分解、重置）合理

### 7.3 进度曲线

- [ ] 新手期（1-7天）：快速获得第一张SSR
- [ ] 成长期（1-4周）：组建完整队伍
- [ ] 成熟期（1-3月）：主力卡牌满级满星
- [ ] 养老期（3月+）：收集养成、PVP竞技
