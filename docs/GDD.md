# 《记忆回收者》游戏设计文档

## 世界观

### 背景设定
"大遗忘"事件后的世界，人类的记忆不再自然消逝，而是凝结成实体化的"记忆碎片"。

你是一名"回收者"——被组织派遣到废墟中，收集这些碎片，还原逝者的故事。

每个碎片都是一个人的一生缩影。当你"养成"这些卡牌，你其实在重新经历他们的人生。

### 核心冲突
- 回收记忆 vs 尊重逝者安宁
- 组织的真实目的逐渐揭露
- 主角自己的记忆也开始碎片化...

---

## 卡牌系统

### 卡牌稀有度
- **普通（灰）**：市井百姓，能力均衡
- **稀有（蓝）**：手艺人、小商贩，有独特技能
- **史诗（紫）**：战士、学者，强力单卡
- **传说（金）**：历史人物、英雄，改变战局
- **神话（红）**：传说中的存在，极为稀有

### 卡牌属性
```
基础属性：
- 生命 HP
- 攻击 ATK
- 防御 DEF
- 速度 SPD

战斗属性：
- 暴击率 CRT
- 暴击伤害 CDMG
- 效果命中 ACC
- 效果抵抗 RES

养成属性：
- 亲密度（解锁剧情）
- 觉醒等级（改变技能）
- 羁绊等级（组队加成）
```

### 卡牌技能
每个角色有3个技能：
- **普攻**：基础攻击，无消耗
- **绝技**：强力技能，消耗能量
- **被动**：常驻效果或触发效果

---

## 养成系统

### 1. 等级系统
- 消耗：记忆粉尘（通用素材）
- 上限：随玩家等级解锁
- 突破：每20级需要特殊材料

### 2. 技能升级
- 消耗：同名卡牌 or 技能书
- 效果：提升伤害/效果数值

### 3. 羁绊系统
- 卡牌之间有"关系链"
- 同时上场触发羁绊效果
- 提升羁绊等级解锁额外剧情

### 4. 觉醒系统
- 满级后可觉醒
- 改变立绘和技能效果
- 大幅提升属性

### 5. 装备系统
- 6个装备位
- 主属性 + 随机副属性
- 套装效果

---

## 战斗系统

### 战斗类型
- **PVE**：主线关卡、日常副本、无尽塔
- **PVP**：竞技场（异步对战）

### 战斗机制
- 回合制，速度决定行动顺序
- 5v5 卡牌对战
- 能量系统：每回合恢复，用于释放绝技
- 站位系统：前排/后排，影响受击优先级

### 战斗元素
- 属性克制：火→风→土→水→火
- 异常状态：燃烧、冰冻、中毒、眩晕等
- 连携攻击：特定卡牌组合触发

---

## 经济系统

### 货币
- **金币**：基础货币，强化、购买道具
- **魂晶**：高级货币，抽卡、刷新商店
- **友情点**：好友互动获得

### 资源获取
- 主线关卡
- 日常副本（金币/经验/材料）
- 挂机收益
- 成就奖励
- 活动奖励

## 主界面架构

### 底部导航栏（5个主要场景）

```
┌─────────────────────────────────────┐
│                                     │
│            当前场景内容              │
│                                     │
├─────────────────────────────────────┤
│  [角色]  [主城]  [探险]  [爬塔]  [设置] │
└─────────────────────────────────────┘
```

### 1. 角色界面
- 主角/主力卡牌展示
- 基础属性面板（HP/ATK/DEF/SPD）
- 战斗属性（暴击/命中/抵抗）
- 6个装备槽位（武器/头盔/护甲/护腿/饰品/戒指）
- 切换角色按钮

### 2. 主城界面
中央是废墟城市背景，分布5个功能性建筑：
- **酒馆（抽卡）**：消耗魂晶抽取新卡牌
- **商店**：购买道具、材料、礼包
- **铁匠铺**：装备强化、精炼、套装打造
- **公会**：公会系统、团队副本
- **训练场**：PVP竞技场

顶部显示：金币、魂晶、体力

### 3. 探险界面（章节关卡）
- 章节选择器（第一章：遗忘之城）
- 50个关卡节点（竖向滚动列表）
- 关卡类型标识：
  - 普通关卡：基础奖励
  - 精英关卡（每5关）：较好奖励
  - Boss关卡（每10关）：稀有奖励
  - 章节Boss（第50关）：章节通关奖励
- 进度显示：已通关数/总关卡数
- 快速扫荡功能（已通关关卡）

**解锁规则**：
- 必须通关上一关才能解锁下一关
- 击败第50关章节Boss后解锁下一章节
- 每关可获得1-3星，影响扫荡收益

### 4. 爬塔界面
- 当前层数显示
- 最高记录显示
- 奖励预览
- 挑战按钮
- 附近楼层列表（±10层）

**爬塔规则**：
- 无上限，无限爬升
- 每层难度递增5%
- 每10层有特殊规则（如敌人属性翻倍）
- 奖励随层数增加而提升
- 可扫荡已通关层数

### 5. 设置界面
- 音量控制（BGM/SFX）
- 震动开关
- 通知设置
- 兑换码输入
- 客服联系
- 隐私政策/用户协议
- 清除缓存
- 版本信息/玩家ID

## 关卡设计

### 第一章：遗忘之城

**背景**：这里曾是最繁华的都市，如今只剩废墟。记忆的碎片在风中飘散。

**关卡结构**：
```
第1-9关：普通敌人，引导教学
第10关：Boss - 废墟守卫
第11-19关：普通敌人，难度提升
第20关：Boss - 记忆掠夺者
第21-29关：普通敌人，加入精英怪
第30关：Boss - 遗忘猎手
第31-39关：高难度普通关卡
第40关：Boss - 破碎骑士
第41-49关：精英关卡为主
第50关：章节Boss - 遗忘之主（解锁下一章节）
```

**关卡类型**：
- **普通**：3个小怪，基础奖励
- **精英**：1精英+2小怪，较好奖励
- **Boss**：Boss+2小怪，稀有奖励
- **章节Boss**：强力Boss+4精英护卫，章节通关奖励

### 爬塔系统

**规则**：
- 每层由3-5个敌人组成
- 敌人等级 = 10 + 层数 × 0.5
- 每10关敌人数量+1
- 每25层特殊规则（敌人属性+50%）
- 每50层特殊规则（玩家能量恢复-50%）

**奖励**：
- 基础：金币 + 经验
- 每5层：额外塔币
- 每10层：稀有卡牌碎片
- 每100层：传说级奖励

## UI设计原则

### 风格
- 废土美学 + 记忆碎片元素
- 色调：暗色底 + 记忆光芒（金/蓝）
- 字体：现代简约 + 适当手写感

### 核心界面
- 主界面：竖屏，底部导航
- 战斗界面：上方敌方，下方己方
- 卡牌详情：全屏展示，可缩放立绘
- 养成界面：树状/线状进度展示

### 交互原则
- 单手操作友好
- 反馈明确（音效+震动）
- 减少加载等待
