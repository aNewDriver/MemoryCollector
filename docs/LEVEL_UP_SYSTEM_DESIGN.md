| 稀有度 | 等级上限 | 每级属性成长 |
|--------|----------|--------------|
| N (普通) | 30级 | 基础属性×1.5 |
| R (稀有) | 40级 | 基础属性×1.8 |
| SR (史诗) | 50级 | 基础属性×2.2 |
| SSR (传说) | 60级 | 基础属性×2.8 |
| UR (神话) | 70级 | 基础属性×3.5 |

## 3. 觉醒系统（高阶养成）

### 3.1 觉醒条件

- 卡牌达到当前稀有度最高星级
- 卡牌等级达到上限
- 完成特定觉醒任务
- 消耗觉醒材料

### 3.2 觉醒阶段

```typescript
interface Awakening {
  stage: number;           // 觉醒阶段 1-3
  requirements: {
    level: number;
    star: number;
    quest: Quest[];
    materials: Material[];
  };
  bonuses: {
    statBonus: number;     // 全属性加成
    newSkill?: string;     // 解锁新技能
    visualUpgrade: boolean; // 外观升级
  };
}
```

**觉醒阶段效果：**

| 觉醒阶段 | 需求 | 属性加成 | 特殊效果 |
|----------|------|----------|----------|
| 1觉 | 满级满星 | 全属性+30% | 解锁觉醒技能 |
| 2觉 | 1觉完成 | 全属性+50% | 技能效果+30% |
| 3觉 | 2觉完成 | 全属性+100% | 解锁终极形态 |

### 3.3 觉醒任务示例

**烬羽（火系SSR）觉醒任务：**
1. 使用烬羽完成100场战斗
2. 使用烬羽造成累计100万伤害
3. 在单场战斗中触发火系克制10次
4. 收集火之精华×100

## 5. 数值策划表结构

### 5.1 卡牌基础数值表

```typescript
interface CardBaseStats {
  id: string;
  name: string;
  rarity: Rarity;
  element: Element;
  
  // 基础属性（1级1星）
  baseHp: number;
  baseAtk: number;
  baseDef: number;
  baseSpd: number;
  baseCrit: number;
  
  // 成长系数
  growthHp: number;
  growthAtk: number;
  growthDef: number;
  growthSpd: number;
  
  // 技能配置
  skills: Skill[];
}
```

### 5.2 数值平衡原则

1. **稀有度差距**：每高一个稀有度，基础属性+20%
2. **等级成长**：高稀有度卡牌后期成长更快
3. **升星收益**：升星比升级收益更高，鼓励深度养成
4. **元素平衡**：不同元素卡牌数值相近，通过克制体现差异
5. **职业差异**：
   - 战士：高攻高血
   - 法师：高攻低防
   - 刺客：高速高暴击
   - 坦克：高血高防
   - 治疗：平衡属性，高治疗加成

### 5.3 数值计算公式

```typescript
// 最终属性计算
function calculateFinalStats(card: Card, level: number, star: number): Stats {
  const base = card.baseStats;
  const levelMultiplier = 1 + (level - 1) * LEVEL_UP_GROWTH.hp;
  const starMultiplier = 1 + (star - 1) * STAR_UP_BONUS.hp;
  const breakthroughMultiplier = 1 + card.breakthroughCount * 0.1;
  const intimacyMultiplier = 1 + card.intimacyLevel * 0.05;
  
  return {
    hp: Math.floor(base.hp * levelMultiplier * starMultiplier * breakthroughMultiplier * intimacyMultiplier),
    // ... 其他属性同理
  };
}
```
