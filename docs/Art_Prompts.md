## Negative Prompt（负面提示词）

```
bright colors, cartoon, anime, 3d render, 
low quality, blurry, deformed hands, extra fingers,
modern city, futuristic technology, clean background
```

---

## 使用说明

1. **Midjourney**: 直接复制Prompt到Discord中生成
2. **Stable Diffusion**: 将Prompt粘贴到正向提示词，Negative Prompt粘贴到负向提示词
3. **分辨率**: 
   - 头像：512x512 或 1024x1024
   - 全身：1024x1820 (9:16)
4. **后期处理**: AI出图后需要统一调色、裁剪、去除背景（如需）

## 命名规范

生成后的图片按以下格式命名：
```
{角色ID}_{类型}.png

例如：
- jin_yu_portrait.png (头像)
- jin_yu_full.png (全身立绘)
- jin_yu_awaken.png (觉醒立绘)
```
