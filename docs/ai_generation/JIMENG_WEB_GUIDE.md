## 批量生成流程

### 步骤1：复制Prompt
1. 复制上面角色的正面Prompt
2. 粘贴到即梦的输入框

### 步骤2：设置参数
1. 选择比例：9:16
2. 选择风格：国风/动漫
3. 负面Prompt粘贴到高级设置

### 步骤3：生成并保存
1. 点击"生成"
2. 等待生成完成（约10-30秒）
3. 选择满意的图片，点击"下载"
4. 重命名为对应的文件名

### 步骤4：放入项目
将下载的图片移动到：
```
assets/resources/images/cards/
```

## 注意事项

1. **风格统一**：同批次生成时使用相同的参数设置
2. **Seed值**：如果支持，记录Seed值以便后续生成类似风格
3. **质量检查**：检查面部是否清晰，特征是否符合描述
4. **分辨率**：全身立绘用1024x1820，头像用512x512
5. **格式**：保存为PNG格式
