# 角色立绘中文Prompt（适配国内AI平台）

## 使用方法

1. 选择你想使用的AI平台（百度文心/阿里通义/腾讯混元/360智绘）
2. 复制对应角色的Prompt
3. 粘贴到平台的输入框中
4. 选择"竖屏"或"9:16"比例，分辨率建议 1024x1820
