# 场景文件索引

生成时间: 3/1/2026, 12:41:53 PM

## 场景列表

- **MainScene** - 主场景 - 包含顶部栏、角色展示、底部导航
- **BattleScene** - 战斗场景 - 5v5回合制战斗界面
- **CardDetailScene** - 卡牌详情 - 角色信息、属性、技能展示
- **GachaScene** - 抽卡场景 - 召唤界面
- **ShopScene** - 商店场景 - 道具购买
- **GuildScene** - 公会场景 - 公会信息、建设
- **InventoryScene** - 背包场景 - 道具管理
- **TowerScene** - 爬塔场景 - 无限之塔挑战

## 使用说明

1. 在 Cocos Creator 中打开项目
2. 双击场景文件即可加载
3. 场景已配置基础 Canvas (750x1334) 和 Camera
4. 根据 UIPrefabConfig.ts 添加UI节点
